Objetivo del Portafolio

El presente portafolio de evidencias tiene como objetivo recopilar, estructurar y presentar de manera sistemática los proyectos y prácticas desarrollados a lo largo del curso. Funciona como una herramienta de evaluación continua que demuestra la evolución de las competencias técnicas adquiridas en el diseño de interfaces de usuario (Frontend) y en la implementación de lógicas de servidor y persistencia de datos (Backend), bajo lineamientos y estándares profesionales de desarrollo de software.

Elementos Aprendidos en la Materia

Desarrollo Frontend y Arquitectura de Diseño

Estructuración semántica utilizando HTML5 y estilización avanzada con CSS3. Integración de sistemas de diseño responsivos y layouts limpios basados en directrices de Figma para asegurar una consistencia visual de alto contraste.

Lógica de Programación y Validación

Manipulación del DOM mediante JavaScript nativo, control de flujos algorítmicos complejos y configuración de expresiones regulares eficaces para la validación estricta de campos en formularios de usuario.

Arquitectura Servidor y APIs (Backend)

Construcción de servidores locales utilizando entornos Node.js y el framework Express. Implementación del patrón de arquitectura Modelo-Vista-Controlador (MVC), mapeo de rutas seguras y manejo de peticiones HTTP (GET, POST, PUT, DELETE).

Persistencia de Datos

Modelado de bases de datos relacionales SQL, conexión persistente mediante controladores del sistema y el diseño e implementación de sistemas CRUD completos para la manipulación e inserción de datos en tiempo real.
